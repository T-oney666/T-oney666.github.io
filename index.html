出现 undefined 的几种情况

1、直接调用函数，但是函数没有返回值。

2、在变量提升（预解析）阶段，只声明未定义，默认值就是 undefined

3、函数定义没有返回值（没有 return）、return 了但 return 后面什么也不带），默认的返回值就是 undefined

4、函数形参不传值，那么这个形参默认就是 undefined

5、对象没有这个属性名。这个属性名默认是 undefined。

6、在 JS 的严格模式下（“use strict”），没有明确的主体，this 指的就是 undefined。

7、数组方法，find 方法

find 方法主要应用与 查找第一个符合条件的数组元素,找到后立马返回,跟 some 和 every 一样
返回通过测试（参数函数内判断）的数组的第一个元素的值

find() 方法返回数组中满足提供的测试函数的第一个元素的值。否则返回 undefined。

find 的参数是一个回调函数，在回调函数中写你要查找元素的条件
数组中的每一个元素都会调用一次函数执行
条件成立为 true 的时候返回该元素,
